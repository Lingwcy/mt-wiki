## 💾 数据更新与保留

::: info 地图更新规则

- 在更新新版本时，会保留生存世界和玩家数据，过旧的资源世界、下界和末地地图将会删除进行更新。
- 服务器会保留两个末地世界，一个末地世界每月1号晚8点刷新，另外一个末地永久不删除。
  :::
